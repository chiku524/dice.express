# Fonts (Comfortaa)

The app uses **Comfortaa** as the primary UI font (see `frontend/src/styles/fonts.css` and `theme.css`).

Place Comfortaa font files here (.ttf format). Required files:
- Comfortaa-Light.ttf
- Comfortaa-Regular.ttf
- Comfortaa-Medium.ttf
- Comfortaa-SemiBold.ttf
- Comfortaa-Bold.ttf
